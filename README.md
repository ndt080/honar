# `Frontend` Honar club
Powerful Honar Sports Club website on React, Tailwind CSS, Radix UI, Zustand and RTK Query

## Features
- 🔥 Written in React with TypeScript
- 💃 Tailwind CSS
- ⚡️ Feature-Sliced Design
- ⚡️ Radix UI primitives
- 🐻 Internationalization
- 🐁 RTK Query
